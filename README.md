## The Vision: Beyond Search

Traditional SEO is a relic of the past. As LLMs become the primary interface for discovery, your brand's "shelf space" has shifted from the Google results page to the **AI's internal weights.**

When a customer asks an AI assistant for a recommendation, they aren't looking for a list of links—they are looking for a definitive answer. If your brand isn't in that answer, you've already lost the sale.

**Pixii AEO Scout** is the diagnostic engine for this new reality. We don't just track keywords; we track **AI Brand Salience.** By cross-referencing AI recommendations against real-world Amazon performance data, we reveal the "AI Blind Spots" where market leaders are invisible to the next generation of shoppers.

---

## Core Capabilities

### Multi-Engine Diagnostic
Scans **Gemini** and **Llama** in parallel to understand how different model architectures perceive your category.

### Amazon Reality Check
Cross-references AI mentions with **Amazon Best Seller Ranks (BSR)**. We find the brands that *should* be mentioned but are being hallucinated away by AI.

### Visibility Grade Card
Produces a high-density, A–F visibility score based on presence, sentiment, and dominance across all engines.

### Obsidian-Grade Analytics
A premium, dark-mode dashboard designed for deep work and high-level strategic decision-making.

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS + Framer Motion
- **Intelligence:** Google Gemini 1.5 + Groq (Llama 3)
- **Data:** Amazon Product Scraping API
- **State:** React Context + Session Storage

## Getting Started

1. **Clone & Install:**
   ```bash
   git clone https://github.com/nidhiatwork01-cmyk/pixii.git
   npm install
   ```

2. **Environment Setup:**
   Create a `.env.local` file:
   ```env
   GEMINI_API_KEY=your_key
   GROQ_API_KEY=your_key
   AMAZON_API_KEY=your_key
   ```

3. **Launch:**
   ```bash
   npm run dev
   ```
